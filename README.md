# Dropbox Clone Backend

Simple dropbox folder hierarchy service - Nodejs &amp; MongoDB

## Running this application

You can run the application by typing:

- `npm install`
- `npm start`

## Testing this application

You can run the application by typing:

- `npm test`

## Documentation

- Eg: `http://localhost:8080/api-doc`
